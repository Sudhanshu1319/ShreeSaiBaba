# Images Directory

Place your images here. The following images are expected:

## Required Images:

1. **baba-murti.jpg** - Main image of Baba's marble murti in the temple
   - Recommended size: 1200x900px or similar aspect ratio
   - Used in: Home page, About page

2. **baba-shirdi-picture.jpg** - Picture of Baba's murthy in Shirdi (mounted on wall)
   - Recommended size: 800x600px or similar
   - Used in: Home page

3. **temple-exterior.jpg** - Exterior view of the temple
   - Recommended size: 1200x800px
   - Used in: Photo Gallery, About page

4. **temple-interior.jpg** - Interior view of the darshan hall
   - Recommended size: 1200x800px
   - Used in: Photo Gallery, About page

5. **aarti-ceremony.jpg** - Aarti ceremony photo
   - Recommended size: 1000x700px
   - Used in: Photo Gallery, Events page

6. **prasadalay.jpg** - Prasadalay (food distribution area)
   - Recommended size: 1000x700px
   - Used in: Photo Gallery

7. **sai-niwas-plans.jpg** - Architectural plans for Sai Niwas
   - Recommended size: 1200x900px
   - Used in: Sai Niwas page

8. **sai-niwas-model.jpg** - 3D model of Sai Niwas
   - Recommended size: 1200x900px
   - Used in: Sai Niwas page

## Photo Gallery Images:

Add more images for the photo gallery in various categories:
- Mandir (temple photos)
- Darshan (Baba's murti photos)
- Events (festivals, celebrations)
- Facilities (prasadalay, resource centre, etc.)
- Community (devotees, gatherings)
- Sai Niwas (construction photos)

## Note:

All images should be optimized for web use. Recommended formats: JPG or WebP.
Keep file sizes reasonable (under 500KB per image when possible) for better loading performance.
