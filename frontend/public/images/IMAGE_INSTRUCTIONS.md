# Image Setup Instructions

## Required Images

Based on the images you provided, please save them with these exact names in the `/frontend/public/images/` folder:

### 1. **baba-murti-throne.jpg**
   - **Description**: The ornate Sai Baba statue on golden throne with garlands
   - **What it shows**: Full statue of Sai Baba seated on elaborate golden throne, wearing blue robe, with golden crown, garlands of red/white/orange flowers, golden canopy above
   - **Used in**: Home page - Main MANDIR section
   - **This is the main temple murti image**

### 2. **baba-murti-portrait.jpg**
   - **Description**: Close-up portrait of white marble Sai Baba statue
   - **What it shows**: White marble bust of Sai Baba with golden crown, red tilak on forehead, colorful garlands (red and yellow), red shawl, golden pendant
   - **Used in**: Home page - Picture of Baba's murthy in Shirdi section
   - **This is the picture mounted on the wall behind the main murti**

## How to Add Images

1. **Save the first image** (ornate golden throne statue) as:
   ```
   frontend/public/images/baba-murti-throne.jpg
   ```

2. **Save the second image** (close-up marble portrait) as:
   ```
   frontend/public/images/baba-murti-portrait.jpg
   ```

3. The website will automatically display these images once they are in place.

## Image Specifications

- **Format**: JPG (recommended) or PNG
- **Quality**: High quality, but optimized for web (under 1MB each if possible)
- **Aspect Ratio**: 
  - `baba-murti-throne.jpg`: Should be landscape (wider than tall) - approximately 4:3 or 16:9
  - `baba-murti-portrait.jpg`: Can be portrait or square - approximately 3:4 or 1:1

## Current Status

The website is ready to display these images. Once you place them in the `/frontend/public/images/` folder with the correct names, they will appear automatically on the home page.
